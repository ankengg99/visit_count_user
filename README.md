
# Show Username and Their count of hiting Api
Simple example of creating Api Using SpringBoot.
Api store user name and count the no visit on Api.


## Prerequisites
Java,Oops,SpringBoot.
## Dependencies
Spring web,Spring dev.
## Build Tool
Maven 
## Install
Maven and Jdk.
